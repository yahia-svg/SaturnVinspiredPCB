# Description
This is an astable multivibrator circuit used to flicker two LED's, activated by holding down the button.
The pcb is supposed to resemble a Rocket.
# Bill of materials
2 capacitors
2 LEDS
2 NPN transistors
2 4.7K resistors
2 220 resistors
1 Push button
1 battery holder
